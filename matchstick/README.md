Matchstick
===

Language:   Python


The project
----
This project is based on a very famous game based on matchsticks.
The game start with a 30 matchsticks long line.
The two players take turns; each player can, take one or several matchsticks.
The losing player is the one to take the last matchstick.

When matchsticks are removed, they must be removed starting from the right.
In case of bad input, you must ask for a number again.


##### OUTPUT
If the user wins, the program must return 1.
If the AI wins, it must return 2.


## USAGE:

```
>> ./matchstick -h
USAGE
	./matchstick max

DESCRIPTION
	max			the maximum number of matchsticks that can be taken out each turn (must be > 0)
```
