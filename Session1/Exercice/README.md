PyScripting
===

Language:   Python


The project
----
This project is to create a script that create an initializated repository for C Projects.

In fact, it will copy files and folders from a template repository and change the binary name generated by the Makefile.


## USAGE:

```
>> ./rinit.py binaryname
```

## PROTIPS:

> You can pass every script you create as an alias in your `.bashrc`/`.zshrc`/...
