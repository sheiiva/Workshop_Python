Workshops Python
===

This repository contains series of 4 python workshops. Its main goal is to understand the usage of [python3](https://docs.python.org/3/index.html) from the bases, to the Object-Oriented Programming paradigm.

## Repository:
* [**Python Documentation**](https://github.com/sheiiva/Workshop_Python/blob/master/documentation.pdf)
* [**Session no.1**](https://github.com/sheiiva/Workshop_Python/tree/master/Session1/README.md) : Python basics (the goal is to understand the bases of python, acquire some vocabulary and key to develop a first python script)
* [**Session no.2**](https://github.com/sheiiva/Workshop_Python/tree/master/Session2/README.md) : Declarative and Imperative python coding style (the goal is to understand the construction of a python program and different basic coding styles)
* [**Session no.3**](https://github.com/sheiiva/Workshop_Python/tree/master/Session3/README.md) : Object-Oriented Programming (the goal here is to understand the OOP paradigm and to get the main keys to create an OOP python code)
* [**Session no.4**](https://github.com/sheiiva/Workshop_Python/tree/master/Session4/README.md) : Tricks and tools (the goal of the last workshop is to get some tips about python3 and conda environment usage)

## PREREQUISITES:
* Python3
* Pip
* Miniconda3
